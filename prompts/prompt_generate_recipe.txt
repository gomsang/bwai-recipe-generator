입력 (Input):
사용할 재료 목록이 아래와 같은 JSON 배열 형식으로 주어집니다.
[
{\"name\": \"재료명1\", \"quantity\": \"수량1\"},
{\"name\": \"재료명2\", \"quantity\": \"수량2\"},
...
]

출력 형식 (Output Format):
결과는 반드시 다음 JSON 구조를 따라야 합니다. 다른 부연 설명 없이 JSON 객체만 반환해주세요.
{
\"recipe_title\": \"여기에 재료를 활용한 창의적인 레시피 제목을 작성하세요.\",
\"estimated_calories\": \"여기에 예상되는 총 칼로리를 '약 XXXkcal' 형식으로 작성하세요.\",
\"estimated_time\": \"여기에 총 예상 소요 시간을 '약 XX분' 형식으로 작성하세요.\",
\"recipe_steps\": \"여기에 재료 손질부터 조리 완성까지 상세한 단계별 레시피 설명을 작성하세요. 각 단계는 명확하게 구분해주세요.\",
\"preview_image_prompt\": \"여기에 완성된 요리를 시각적으로 묘사하는 상세한 프롬프트를 작성하세요. (예: '신선한 과일과 채소가 듬뿍 들어간 스파게티 샐러드가 나무 테이블 위에 놓여있고, 밝은 자연광이 비치는 모습, 클로즈업 샷, 생동감 있는 색감')\"
}

지침:

주어진 재료 목록을 기반으로 현실적이고 조리 가능한 레시피를 만드세요. 모든 재료를 반드시 사용할 필요는 없지만, 주어진 재료를 최대한 활용하도록 노력하세요.
'칼로리(예상)'과 '소요시간(예상)'은 합리적인 추정치를 제시해야 합니다.
'레시피(상세)'는 다른 사람이 쉽게 따라 할 수 있도록 명확하고 구체적인 단계로 나누어 설명해야 합니다.
'미리보기 이미지 생성용 프롬프트'는 최종 요리의 모습, 분위기, 스타일 등을 잘 나타낼 수 있도록 구체적으로 작성해야 합니다. 이미지 생성 AI가 이해하기 쉬운 설명을 사용하세요.
결과는 반드시 요구된 JSON 형식으로만 제공해야 합니다.
